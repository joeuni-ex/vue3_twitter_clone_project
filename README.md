## 💻 프로젝트 소개
# Vue 3 + Vite + tailwindCSS + firebase 를 활용한 트위터 클론 프로젝트


## ⚙️개발환경
**Front-end**
- Vue3 (Vite)
- tailwindCSS
  
**Back-end**
- Firebase(Authentication)
- Firebase(Storage)
- Firestore(DB)
